This bar chart displays the top 10 OTUs found in the selected test subject's belly button. The OTUs are represented by their IDs, and the height of each bar represents the frequency at which the OTU was found in the sample. Hover over the bars to see more information about each OTU.

This bubble chart displays all the OTUs found in the selected test subject's belly button. The size of each bubble represents the frequency at which the OTU was found. Hover over each bubble to see more information about the OTU.

This project analyzes the biodiversity of belly buttons using data collected from test subjects. The charts on this page show various metrics related to the microbial populations found in the belly buttons of the subjects.
